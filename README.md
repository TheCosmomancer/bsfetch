# Fetch BS system information (because why not)

bsfetch is a fun program for displaying satire system information. Despite that, the software and hardware mentioned here is (for the most part) real and technically usable. Go take a look for yourself.

## Usage
Just clone the repo with:
```git clone https://github.com/cosmomaniac/bsfetch.git```

and run with python:
```python bsfetch.py```

no package imports or (to my knowledge) new fetures of python have been used yet; so it should be rather compatible with older versions of python3. I have no clue as to how far back though.
